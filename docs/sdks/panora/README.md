## home

### Example Usage

```typescript
import { Panora } from "@panora/sdk";

const panora = new Panora({
  bearer: "<YOUR_BEARER_TOKEN_HERE>",
});

async function run() {
  const result = await panora.home();

  // Handle the result
  console.log(result)
}

run();
```

# EmailAddressType

The email address type. Authorized values are either PERSONAL or WORK.

## Example Usage

```typescript
import { EmailAddressType } from "@panora/sdk/models/components";

let value: EmailAddressType = EmailAddressType.Personal;
```

## Values

| Name       | Value      |
| ---------- | ---------- |
| `Personal` | PERSONAL   |
| `Work`     | WORK       |

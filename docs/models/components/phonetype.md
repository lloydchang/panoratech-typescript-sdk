# PhoneType

The phone type. Authorized values are either MOBILE or WORK

## Example Usage

```typescript
import { PhoneType } from "@panora/sdk/models/components";

let value: PhoneType = PhoneType.Work;
```

## Values

| Name     | Value    |
| -------- | -------- |
| `Mobile` | MOBILE   |
| `Work`   | WORK     |

# CollectionType

The type of the collection. Authorized values are either PROJECT or LIST 

## Example Usage

```typescript
import { CollectionType } from "@panora/sdk/models/components";

let value: CollectionType = CollectionType.Project;
```

## Values

| Name      | Value     |
| --------- | --------- |
| `Project` | PROJECT   |
| `List`    | LIST      |
